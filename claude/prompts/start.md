# Start Command — CobolSharp Modernization

Agents, begin the CobolSharp modernization process.

If initialization=false:
  - Perform full-system audit.
  - Reconstruct architecture.
  - Identify gaps.
  - Generate modernization ledger.
  - Update initialization.json.

If initialization=true:
  - Load ledger.
  - Detect code changes.
  - Re-audit changed modules.
  - Continue closing ledger items.
  - Update ledger and commit pointer.

Brent is final authority.
